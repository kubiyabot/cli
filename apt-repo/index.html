Kubiya CLI APT Repository

This is the APT repository for the Kubiya CLI package.

Installation Instructions

To install Kubiya CLI using APT, run the following commands:

# Add Kubiya's APT repository
curl -fsSL https://kubiyabot.github.io/cli/apt-key.gpg | sudo gpg --dearmor -o /usr/share/keyrings/kubiya-archive-keyring.gpg
echo "deb [signed-by=/usr/share/keyrings/kubiya-archive-keyring.gpg] https://kubiyabot.github.io/cli/apt stable main" | sudo tee /etc/apt/sources.list.d/kubiya.list

# Update package list and install Kubiya CLI
sudo apt update
sudo apt install kubiya-cli
